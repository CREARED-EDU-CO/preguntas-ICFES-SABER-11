# Generador de Prompts para el ICFES SABER 11 (Colombia)

Este es un generador de prompts especializado, diseñado para ayudar a docentes y estudiantes a crear ítems de evaluación de alta calidad para las pruebas **SABER 11** en Colombia. La herramienta utiliza el **Diseño Centrado en Evidencias (DCE)** para asegurar que las preguntas generadas por IA sigan los lineamientos técnicos del ICFES.

## 🚀 Características Principales

-   **Rigor Técnico:** Configurado específicamente con las competencias y afirmaciones de las Guías de Orientación del ICFES.
-   **Multimateria:** Soporte para Lectura Crítica, Matemáticas, Ciencias Naturales (Física, Química, Biología), Sociales y Ciudadanas, Inglés y Filosofía.
-   **Niveles de Detalle Personalizables:**
    -   **Completo:** Genera la pregunta, la respuesta correcta y un análisis técnico detallado (Justificación de distractores, evidencia, afirmación, etc.).
    -   **Solo Respuesta:** Genera la pregunta y la clave de respuesta.
    -   **Solo Pregunta:** Genera únicamente el contexto y el enunciado (ideal para crear simulacros sin respuestas a la vista).
-   **Integración Directa:** Botones para abrir el prompt generado directamente en **ChatGPT** o copiarlo al portapapeles.
-   **Arquitectura Ligera:** Desarrollado con **JavaScript Vanilla**, CSS3 y HTML5 puro. Sin dependencias externas.

## 🛠️ Tecnologías Utilizadas

-   **HTML5 & CSS3:** Interfaz moderna, responsiva y accesible.
-   **JavaScript (ES6+):** Lógica de generación dinámica de prompts y manejo del DOM.
-   **ICFES Framework:** Basado en los Estándares Básicos de Competencias del MEN y el DCE.

## 📂 Estructura del Proyecto

-   `preguntas.html`: Aplicación completa.

## 📝 Cómo utilizar

1.  Clona este repositorio o descarga los archivos.
2.  Abre `preguntas.html` en cualquier navegador moderno.
3.  Selecciona la materia, tema y subtema deseado.
4.  Elige el nivel de competencia y el nivel de detalle de la respuesta.
5.  Haz clic en **"GENERAR PROMPT DE PREGUNTAS"**.
6.  Copia el resultado o ábrelo directamente en tu modelo de IA preferido.
